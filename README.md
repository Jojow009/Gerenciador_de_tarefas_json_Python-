# 📋 Gerenciador de Tarefas em Python

Um aplicativo de desktop simples e leve para gerenciamento de tarefas (To-Do List), desenvolvido em Python com interface gráfica e salvamento automático de dados.

## ✨ Funcionalidades

* **Interface Gráfica Amigável:** Desenvolvida usando a biblioteca nativa `Tkinter`.
* **Persistência de Dados (JSON):** As tarefas são salvas automaticamente no arquivo `tasks.json`. Você não perde suas anotações ao fechar o programa!
* **Histórico de Ações:** Sistema integrado de **Desfazer (Undo)** e **Refazer (Redo)** para recuperar tarefas removidas por engano.
* **Leve e Rápido:** Não requer instalação de bancos de dados complexos ou bibliotecas externas pesadas.

## 🛠️ Tecnologias Utilizadas

* **Python 3**
* **Tkinter** (Interface Gráfica nativa do Python)
* **Módulo JSON** (Manipulação e salvamento de dados)
